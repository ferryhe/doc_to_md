# ============================================================
# requirements-recommended-pdf.txt
#
# Current recommended PDF environment for this repository.
#
# Keeps the engines we currently recommend:
#   - local
#   - markitdown
#   - opendataloader
#   - docling
#   - mistral
#   - mathpix
#
# Notes:
#   - local, mistral, and mathpix come from the base package
#   - mistral still needs MISTRAL_API_KEY
#   - mathpix still needs MATHPIX_APP_ID and MATHPIX_APP_KEY
#   - opendataloader still needs Java 11+ on the system
#   - mathpix is mainly for handwritten formulas and image-like math pages
#
# Install:
#   pip install -r requirements-recommended-pdf.txt
#
# Equivalent direct command:
#   pip install -e ".[markitdown,docling,opendataloader]"
# ============================================================

-e .[markitdown,docling,opendataloader]
